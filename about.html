Our Mission

Code for Long Island is a Code for America Brigade, or local chapter, focused on improving the lives of residents of Long Island. Code for Long Island is made up of individuals who are interested in changing Long Island for the better.

Code for Long Island is part of the Brigade Network of Code for America. Learn more about Code for America here.

We work with local government to fix local services, utilize open data to improve citizen knowledge, and work for the betterment of all. We are just getting started, so there are plenty of opportunities to use your skills.

We always need more developers, but we welcome designers, content creators, copywriters, storytellers, researchers, project managers, marketing specialists, government enthusiasts, or anyone else! So even if you are not a tech type, we can still have opportunities for you to contribute.

So come by and become part of a movement to improve our community!
